# שיחות דוגמה / Call Demos

תיקייה זו מכילה קבצי MP3 של שיחות דוגמה שמוצגים באתר.

## כיצד להוסיף שיחה חדשה

1. העלה קובץ MP3 עם השם: `call-<id>.mp3`
   - לדוגמה: `call-1.mp3`, `call-2.mp3`, `call-3.mp3`

2. ודא ש-`<id>` תואם ל-`id` שהוגדר בקובץ `content/site.he.json` תחת `callDemos.calls`

3. האתר יטען את הקבצים אוטומטית לפי שם הקובץ

## דוגמה

אם יש ברשימה ב-JSON:
```json
{
  "id": "call-1",
  "title": "תיאום תור במרפאה",
  "description": "..."
}
```

אז צריך להעלות קובץ בשם: `call-1.mp3`

## הערות
- התיקייה ריקה כברירת מחדל
- קבצים חסרים לא ישברו את האתר - יוצג הודעת "שיחה לא זמינה עדיין"
- מומלץ גודל קובץ מינימלי של 30KB
