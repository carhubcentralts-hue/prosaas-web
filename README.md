# ProSaaS Web

אתר תדמית ודף נחיתה ל-ProSaaS - מערכת CRM + בוט WhatsApp + בוט שיחות בזמן אמת.

## מבנה הפרויקט

```
/landing    # Next.js אתר תדמית
```

## הרצה מקומית

```bash
cd landing
npm install
npm run dev
```

## פריסה

האתר מוכן לפריסה ב-Vercel. ראו [landing/README.md](landing/README.md) להוראות מפורטות.
